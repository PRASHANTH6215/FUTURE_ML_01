# Sales Demand Forecasting using Machine Learning

This project predicts sales using a machine learning model built using Python.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Features
- Data cleaning
- Feature engineering
- Linear regression model
- Sales prediction visualization

## Dataset
Sample Superstore dataset containing order and sales information.

## Output
The model predicts future sales based on date features.
